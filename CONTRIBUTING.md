# Contributing | 贡献指南

感谢你对本项目的关注！欢迎提交 Issue 和 Pull Request。

## 如何贡献

### 1. Fork 本仓库
点击右上角 **Fork** 按钮，将仓库复制到你的账号下。

### 2. Clone 到本地
```bash
git clone https://github.com/<your-username>/Windows_Auto_Inspection.git
cd Windows_Auto_Inspection
```

### 3. 创建分支
```bash
git checkout -b feature/your-feature-name
```

### 4. 修改代码
- 修改或新增 PowerShell (.ps1) 脚本
- 确保脚本在目标环境可正常运行
- 添加必要的注释说明

### 5. 提交并推送
```bash
git add .
git commit -m "Add: description of your change"
git push origin feature/your-feature-name
```

### 6. 创建 Pull Request
回到 GitHub，点击 **Compare & pull request**，填写说明后提交。

## 规范
- 提交前请在目标环境测试通过
- 提交信息使用中文或英文均可
- 一个 PR 只做一件事
